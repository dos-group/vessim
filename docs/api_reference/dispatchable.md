# Dispatchable

Controllable energy resources whose power is allocated by a
[Dispatch Policy](dispatch_policy.md). Subclass [`Dispatchable`](#vessim.Dispatchable)
to model any device with a controllable power setpoint. See the
[Dispatchables and Dispatch Policies](../concepts/dispatchables.md) concept page for
usage examples and a guide to writing custom dispatchables.
